# babel-eslint 8版本以上会报token.type.endsWith is not a function vue eslint错误

错误原因：babel-eslint包已经不维护了

解决办法：

（1）、换成@babel/eslint-parser包；

（2）、在.eslintrc.js文件添加

parserOptions: { parser: '@babel/eslint-parser' }
